# Neatnik Detailing

Family-owned car detailing in Fairfield, Maine — serving Fairfield and surrounding Mid-Maine communities.

**Website:** https://neatnikdetailing.github.io/neatnikdetailing/

## About this repository

This repository hosts the public website for Neatnik Detailing. The site is currently an active mockup that is being refined into the business’s official website home.

Published contents are for viewing and hosting the live site. See [LICENSE](./LICENSE) for proprietary terms.

## What’s on the site

- Detailing packages (interior, exterior, and full detail)
- Starting prices and what’s included
- Service area information
- Gallery / before-and-after section
- About Neatnik
- Contact and booking request form
